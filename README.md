# Expense Tracker

## Project Description
A simple expense tracker which takes in 3 input fields: Description, Amount and Category(Groceries, Utility, Entertainment) and add's it to the list of expenses along with the grad total. Furthermore, we can filter the list based on category and delete items from the list. Built in react.

## Installation
1. Clone the repository.
2. Navigate to the project directory.
3. Run `npm install` to install dependencies.

## Usage
1. Run the application with `npm run dev`.
2. Open your browser and navigate to `http://localhost:5173` or the local host link shown under Local.
